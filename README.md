# CineScope 🎬

Built this to learn how REST APIs work on Android. It pulls popular movies from TMDB and shows them in a list — poster, title, rating, synopsis.

## Stack

Kotlin, Retrofit, OkHttp, Picasso, Coroutines, ViewBinding
